Kosakata & Percakapan Bahasa Jawa

Materi ini membantu kamu memahami kosakata dasar, pola kalimat, serta percakapan sehari-hari dalam Bahasa Jawa (Ngoko).

1. Salam Sehari-hari

Jawa Indonesia Penggunaan
Sugeng enjingSelamat pagiPagi hari
Sugeng siangSelamat siangSiang
Sugeng sontenSelamat soreSore
Sugeng daluSelamat malamMalam
Piye kabarmu?Apa kabar?Menanyakan kondisi
Contoh: Sugeng enjing, piye kabarmu?

2. Perkenalan Diri

Pola dasar:

Jenengku + (nama)
Aku saka + (asal)
Umurku + (umur)

Contoh:

Jenengku Dwiki, aku saka Surabaya. Umurku 20 taun.

3. Kosakata Keluarga

BapakAyah
IbuIbu
KakangKakak laki-laki
MbakyuKakak perempuan
AdikAdik
SimbahKakek/Nenek

Contoh: Iki bapakku → Ini ayah saya

4. Aktivitas Harian

  • Mangan → Makan
  • Ngombe → Minum
  • Turu → Tidur
  • Sinau → Belajar
  • Lunga → Pergi
Aku sinau saben dina → Saya belajar setiap hari

5. Pola Kalimat Dasar

Struktur:

S (Subjek) + P (Predikat) + O (Objek)
  • Aku mangan nasi → Saya makan nasi
  • Kowe ngombe banyu → Kamu minum air

6. Kalimat Tanya

ApaApa
SapaSiapa
PiraBerapa
Ning endiDi mana

Contoh: Kowe arep ning endi?

7. Kalimat Negasi

Ora = Tidak

  • Aku ora mangan → Saya tidak makan
  • Dheweke ora teka → Dia tidak datang

8. Percakapan Sederhana

A: Sugeng enjing!

B: Sugeng enjing, piye kabarmu?

A: Apik, matur nuwun.

B: Aku uga apik.

Latihan

Terjemahkan:

  • Saya makan nasi
  • Kamu pergi ke sekolah
  • Apa kabar?
